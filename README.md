## View DynamoDB Local tables with DyanmoDB Admin GUI
```bash

// For Windows
set DYNAMO_ENDPOINT=http://localhost:8001
dynamodb-admin

// For Mac/Linux
DYNAMO_ENDPOINT=http://localhost:8001 dynamodb-admin
```
